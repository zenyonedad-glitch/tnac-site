# TNAC Corporate Site

Minimal static corporate website for Apple Developer Program organization enrollment readiness.

## Files

- `index.html`: company overview and contact page
- `privacy.html`: minimal corporate-site privacy policy
- `styles.css`: responsive styling
- `assets/mark.svg`: simple TNAC mark / favicon
- `robots.txt`: crawler directive

## Intended Public URL

- `https://tnac-ai.com/`

## Before Publishing

1. Replace local links with final absolute URLs if the hosting platform requires them.
2. After deployment, verify that `https://tnac-ai.com/` and `https://tnac-ai.com/privacy.html` load without a login screen or parking page.
